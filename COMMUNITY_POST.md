# BYD 차량 자동 잠금/해제 iOS 앱 만들었습니다 (비공식, 오픈소스)

안녕하세요.

BYD 전기차 오너분들을 위해 **블루투스 신호 기반 자동 잠금/해제 iOS 앱**을 만들었습니다.
기존에 안드로이드용으로 공개된 [BydAutoLock](https://github.com/PoorGrammerA/BydAutoLock) 앱을 iOS Swift로 완전히 새로 구현한 비공식 오픈소스 프로젝트입니다.

GitHub: https://github.com/GeyuongGongPark/BydAutoLock-iOS

---

## 어떤 앱인가요?

iPhone을 주머니에 넣고 차량에 가까이 가면 **자동으로 잠금 해제**, 멀어지면 **자동으로 잠금**되는 앱입니다.
BYD 차량의 블루투스 신호 강도(RSSI)를 지속적으로 측정하여 차량과의 거리를 판단합니다.

---

## iPhone 앱 기능

### 자동 제어
- 블루투스 신호가 설정한 임계값 이상이면 **자동 잠금 해제**
- 블루투스 신호가 설정한 임계값 이하면 **자동 잠금**
- 신호가 갑자기 끊겨도 **2분 대기 후** 잠금 (오작동 방지)
- EMA + 선형 회귀 알고리즘으로 **신호 노이즈 필터링**

### 차량 상태 확인
- 현재 잠금/열림 상태
- 배터리 잔량 (%)
- 주행 가능 거리 (km)
- 실내 온도
- 에어컨 켜짐/꺼짐 상태

### 수동 제어
- 앱에서 직접 잠금/해제 버튼

### 에어컨 자동 제어
- 잠금 해제 시 에어컨 자동 켜기 (목표 온도 설정 가능)
- 잠금 시 에어컨 자동 끄기

### 배터리 절약
- 차량에서 150m 이상 멀어지면 블루투스 스캔 자동 중단 (지오펜싱)
- 5분간 이동이 없으면 스캔 일시 중단, 움직이면 자동 재개

### 백그라운드 동작
- 앱을 완전히 종료하거나 화면을 꺼도 자동 잠금/해제 동작
- 지역 서버 15개국 지원: 한국/유럽/일본/싱가포르/호주/브라질/멕시코 등

## 홈 화면 / 잠금 화면 위젯

| 위젯 종류 | 내용 |
|---|---|
| 작은 위젯 (2×2) | 서비스 ON/OFF · 잠금 상태 · 배터리 % |
| 중간 위젯 (4×2) | 서비스 · 잠금 · 배터리 · 블루투스 신호 |
| 잠금 화면 원형 | 잠금 상태 아이콘 |
| 잠금 화면 직사각형 | 간단 상태 요약 |

---

## Apple Watch 앱 기능

iPhone과 블루투스로 연결된 Apple Watch에서도 바로 사용 가능합니다.

- 차량 잠금/열림 상태 확인
- 서비스 ON/OFF 상태 확인
- 배터리 % 확인
- **손목에서 바로 잠금/해제 버튼** 조작
- 서비스 시작/중지

---

## Watch 페이스 컴플리케이션

시계 화면에서 바로 차량 상태를 확인하고 제어할 수 있습니다.

| 컴플리케이션 | 기능 |
|---|---|
| 원형 | 잠금 상태 아이콘 + 배터리 % |
| 직사각형 | 서비스 ON/OFF + 잠금 상태 + 배터리 + **잠금/해제 버튼** |
| 코너 | 잠금 아이콘 + ON/OFF |

직사각형 컴플리케이션에서 잠금/해제 버튼을 탭하면 Watch 앱이 열리며 즉시 동작합니다.

## 설치 방법 (개발자 계정 없이 가능)

iOS 앱은 앱스토어에 등록되지 않았습니다. 직접 빌드하여 설치해야 합니다.

### 필요한 것
- Mac (Xcode 설치)
- 무료 Apple ID (개발자 계정 불필요)
- AltStore 또는 Sideloadly

### 순서
1. GitHub에서 소스코드 다운로드
2. Xcode로 빌드 (Product → Archive)
3. IPA 파일 추출
4. AltStore / Sideloadly로 iPhone에 설치
5. **설정 → 일반 → VPN 및 기기 관리**에서 신뢰 허용

> 무료 Apple ID는 7일마다 재설치 필요

---

## 초기 설정 방법

1. 앱 실행 → **설정 → BYD 계정 설정**
   - BYD 앱과 동일한 이메일/비밀번호 입력
   - 차량 제어 PIN 입력
   - 지역 선택 (한국은 KR)
   - 로그인 버튼 탭

2. **설정 → 블루투스 기기 설정**
   - 차량 블루투스를 켠 상태에서 스캔
   - 목록에서 차량 선택

3. **설정 → RSSI 임계값 설정**
   - 잠금 해제 임계값: 기본 -70 dBm (차량 가까이)
   - 잠금 임계값: 기본 -85 dBm (차량 멀어짐)
   - 직접 차량 옆에서 테스트하며 조정 권장

4. 메인 화면에서 **자동 잠금 서비스 토글 ON**

## 주의 사항

- 비공식 앱으로 BYD 공식 앱과 무관합니다
- 블루투스 신호는 환경(건물, 전파 간섭)에 따라 달라질 수 있습니다
- 처음 사용 시 임계값을 직접 조정하며 테스트하는 것을 강력히 권장합니다
- iOS 백그라운드 제한으로 포그라운드에서 더 빠르게 반응합니다
- Apple Watch 제어는 iPhone이 근처에 있어야 동작합니다 (셀룰러 Watch 제외)
