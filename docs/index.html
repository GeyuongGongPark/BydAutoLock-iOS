기능

필요한 건 다 있습니다

BYD 공식 앱 API를 활용해 실제 차량 제어까지 연결합니다.

🔓

자동 잠금 해제

BLE RSSI가 임계값 이상이 되면 자동으로 차량 잠금을 해제합니다.

🔒

자동 잠금

스마트폰이 멀어지면 자동으로 잠급니다. 차 키를 꺼내지 않아도 됩니다.

📡

EMA 신호 필터링

지수 이동 평균과 선형 회귀 이상값 감지로 흔들림 없는 안정적인 거리 측정.

❄️

에어컨 자동 제어

잠금 해제 시 에어컨을 자동으로 켜고, 잠금 시 끌 수 있습니다. 온도·풍량 설정 가능.

📍

주차 위치 기록

차량 GPS를 주기적으로 조회해 마지막 주차 위치를 지도로 확인합니다.

⌚

Apple Watch 지원

워치에서 잠금 상태 확인 및 수동 잠금/해제 제어가 가능합니다.

🔲

홈 화면 위젯

잠금 상태와 배터리를 홈 화면·잠금 화면 위젯으로 한눈에 확인.

🗺️

지오펜스

주차 위치 근처에서만 BLE 스캔을 활성화해 배터리를 절약합니다.

작동 방식

어떻게 동작하나요?

차량의 BLE 신호 세기(RSSI)를 측정해 거리를 추정합니다.

1

BLE 신호 감지

앱이 백그라운드에서 차량 BLE 기기에 연결(connect)해 3초마다 RSSI를 측정합니다. 백그라운드에서도 안정적으로 작동합니다.

2

거리 판단

측정된 RSSI에 EMA 필터를 적용해 노이즈를 제거합니다. 설정한 임계값과 비교해 접근/이탈을 판단합니다.

3

차량 제어 명령

BYD 공식 API를 통해 잠금/해제 명령을 전송합니다. 명령 결과는 알림으로 확인할 수 있습니다.

4

자동 에어컨 (선택)

잠금 해제 시 설정한 온도로 에어컨을 자동으로 켭니다. 타기 전에 이미 쾌적한 온도가 됩니다.

설치

설치 방법 선택

세 가지 방법 중 편한 방법으로 설치하세요.

AltStore 또는 Sideloadly로 IPA 설치

Apple 개발자 계정 없이 무료 Apple ID만으로 설치할 수 있습니다. 7일마다 재서명이 필요합니다.

1

AltStore 설치

Mac 또는 Windows PC에 AltStore를 설치합니다. 또는 Sideloadly를 사용해도 됩니다.

2

IPA 파일 다운로드

GitHub Releases 페이지에서 최신 BydAutoLock.ipa를 다운로드합니다.

→ GitHub Releases에서 다운로드

3

iPhone에 설치

iPhone을 Mac/PC에 연결한 후 AltStore에서 IPA 파일을 선택해 설치합니다.

4

신뢰 설정

설정 → 일반 → VPN 및 기기 관리에서 개발자 앱을 신뢰합니다.

⚠️

무료 Apple ID는 7일마다 재서명이 필요합니다. AltStore의 자동 갱신 기능을 활용하거나, 유료 Apple Developer 계정($99/년)을 사용하면 365일 유효합니다.

TestFlight 베타 설치

Apple의 공식 베타 배포 플랫폼입니다. 설치와 업데이트가 가장 간편합니다.

1

TestFlight 앱 설치

App Store에서 TestFlight를 설치합니다.

2

초대 링크로 참여

아래 초대 링크를 iPhone에서 열면 TestFlight가 자동으로 실행됩니다.

🔗 TestFlight 초대 링크는 준비 중입니다

3

설치 및 업데이트

TestFlight에서 "설치" 버튼을 탭합니다. 새 버전이 나오면 TestFlight에서 알림으로 안내합니다.

ℹ️

TestFlight 베타는 최대 90일간 유효합니다. 현재 베타 테스터를 모집 중입니다. GitHub Issues에서 참여를 요청해주세요.

Xcode로 직접 빌드

소스 코드를 수정하거나 직접 빌드하고 싶은 개발자를 위한 방법입니다.

1

저장소 클론

git clone https://github.com/GeyuongGongPark/BydAutoLock-iOS.git cd BydAutoLock
2

Xcode에서 프로젝트 열기

open BydAutoLock.xcodeproj
3

개발 팀 서명 설정

Xcode에서 BydAutoLock 타겟 선택 → Signing & Capabilities → Team을 본인의 Apple ID로 변경합니다.

4

iPhone 연결 후 빌드

iPhone을 연결하고 대상 기기를 선택한 후 ⌘R로 빌드 및 설치합니다.

5

신뢰 설정

설정 → 일반 → VPN 및 기기 관리에서 개발자 앱을 신뢰합니다.

⚠️

Xcode 15 이상, macOS 13 이상이 필요합니다. 무료 Apple ID로 빌드 시 7일마다 재빌드가 필요합니다.
